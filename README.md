# CS-370-Portfolio
Course portfolio for CS-370: Emerging Trends in Computer Science, including the Pirate Intelligent Agent
## Project Overview

This repository contains my Pirate Intelligent Agent project for **CS-370: Emerging Trends in Computer Science**. The goal of this project was to apply reinforcement learning and neural networks to solve a pathfinding problem in a treasure hunt game environment.

The intelligent agent (the pirate) learns how to navigate a maze and locate the treasure by interacting with the environment, receiving rewards, and improving its decision-making over time using deep Q-learning.


## Work Completed on This Project

For this project, I was provided with starter code that included the basic game environment, maze layout, and supporting classes needed to run the treasure hunt simulation. This starter code handled the structure of the game, state representation, and initial setup for the agent.

I implemented the core logic for the intelligent agent myself. This included configuring and training the deep Q-learning model, defining how the agent selects actions, updating Q-values based on rewards, and balancing exploration and exploitation. I also worked with neural network components to allow the agent to learn optimal paths through repeated training episodes.


## Connection to the Field of Computer Science

Computer scientists design, build, and improve systems that solve real-world problems through computation. This work matters because intelligent systems are increasingly used in areas such as automation, data analysis, robotics, and artificial intelligence. Projects like this demonstrate how algorithms can learn from experience instead of relying solely on hard-coded logic.

When approaching a problem as a computer scientist, I start by understanding the problem space, identifying constraints, and selecting an appropriate algorithm or model. In this case, reinforcement learning was well-suited because the agent needed to learn through trial and error rather than being given a predefined solution.


## Ethical Responsibilities

As a computer scientist, I have an ethical responsibility to consider how systems affect both users and organizations. Intelligent agents must be designed responsibly to avoid unintended consequences, bias, or misuse. Ensuring transparency, reliability, and fairness in AI-driven systems is critical.

In this project, ethical considerations include ensuring that learning algorithms behave predictably, do not reinforce harmful patterns, and are used in appropriate contexts. Responsible development helps build trust in intelligent systems and ensures they provide value without causing harm.
